---

You are a frontend engineer building a polished static web app.

Create a React + Vite website using Bootstrap and minimal custom CSS for a light-themed vocational education platform (E-Kaushalaya Hub) by Mindstix Foundation Trust.

Requirements:
- Homepage with:
  - hero section
  - presentation of Mindstix Foundation Trust's contribution to ITI students
  - trade selection presented as a 3-column bento-style grid
  - footer containing all required contact information from mindstixfoundation.org
- Trade selection should navigate to a module page.
- Module page should display modules in a responsive 3-column grid and a list view on mobile.
- From the module page, users should navigate to a video section.
- The video section should render videos in a 3-column grid on desktop and a list view on mobile.
- Video data must come from a CSV source with columns:
  Playlist ID | Trade | Year | Module | Title | Description | YouTube video link | Google Forms
- On the module page, include a "take test" Google Forms button when the CSV contains a Google Forms link.
- Do not embed tests inside the videos.
- Clicking a video should open a dedicated video detail page with:
  - embedded YouTube video
  - title
  - description
  - assessment link from Google Forms
  - a sidebar showing next module videos for the same trade/module
- Only display fields that exist in the CSV row; omit missing values gracefully.
- Implement a video resume feature using browser storage so playback continues from the last paused position.
- Add search functionality for modules and videos.
- Add a persistent global feedback button that redirects to https://forms.gle/fTqD15CNxV2U1vd8A.
- Add a global referral button or note for "Nimi" that redirects to https://bharatskills.gov.in/.
- Add a global "Mock test 'Nimi'" button linking to https://play.google.com/store/apps/details?id=com.nimi.nimimocktest&hl=en&gl=US.
- After the user completes 5-6 videos, show a feedback pop-up titled "Feedback" with a short description and two buttons: "Not, now" and "Yes, now".
- Display the feedback pop-up at the bottom right corner on desktop and full-width bottom on mobile.
- Keep the design simple, easy to use, and aligned with Mindstix logo colors in a light theme.
- Use React + Vite, Bootstrap, and vanilla CSS only.

Deliverables:
- Project structure and code for all pages, components, and data loading.
- Routing for homepage, trade modules, video grid, and video detail pages.
- Responsive mobile behavior with list views for modules and videos.
- Local storage support for resume playback.
- Example CSV data and logic to parse it.

Example prompt use:
- "Generate the full React + Vite app for the Mindstix ITI digital education platform with trade selection, modules, YouTube videos, Google Forms tests, and resume playback."
